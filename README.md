3. **[Namma Traffic](https://github.com/saminadamn/namma-traffic)**: Real-time traffic intelligence platform for Bengaluru — CatBoost incident prediction (<5ms inference), incident-aware OSRM routing, async FastAPI backend, Next.js dashboard. Built for Flipkart GRiD Gridlock 2.0.
4. **BAH Satellite Radiation Dashboard**: A directional (east/west) electron flux forecasting and anisotropy-based operational risk dashboard for satellites, using TFT and LightGBM models on GOES, OMNI, and GSAT/GRASP data. Built for ISRO's Bharatiya Antariksh Hackathon 2026.
5. **[DHANSETU](https://github.com/saminadamn)**: A full-stack, ML-assisted financial inclusion platform automating NBCFDC loan eligibility assessment. **Smart India Hackathon 2025 Grand Finalist.**
6. **[ADHD Behavioral Modeling Framework](https://github.com/saminadamn/ADHD-behavioral-attention-score)**: A LangGraph multi-agent system extracting behavioral signals from 10,000+ conversational data points to classify attention states, built under faculty supervision.
7. **Memory Bridge**: A voice-to-story NLP tool that helps Alzheimer's patients and caregivers reconstruct and preserve personal memories.
8. **Urban Heat Island Monitoring Platform**: A geospatial monitoring system that tracks and visualizes urban heat island effects using satellite and sensor data.
9. **Helipad Detection CNN**: A convolutional neural network for helipad detection from aerial imagery, achieving 92.5% accuracy.
