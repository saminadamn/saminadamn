## Works

### Projects
1. **[Namma AI](https://github.com/saminadamn/namma-traffic)**: Real-time, full-stack urban traffic intelligence platform for Bengaluru built for Flipkart Gridlock 2.0. Async FastAPI + PostgreSQL backend, a CatBoost model with <5ms inference for road-closure prediction, incident-aware OSRM routing (<3s route computation), and live WebSocket telemetry to a Next.js dashboard. [Live demo](https://namma-traffic-virid.vercel.app/)
2. **[ADHD Behavioral Modeling AI Framework](https://github.com/saminadamn/ADHD-behavioral-attention-score)**: A LangGraph multi-agent research framework processing 10,000+ conversational data points to extract behavioral signals and classify attention states, with custom pipelines accelerating longitudinal tracking by 40%.
3. **[DHANSETU](https://github.com/saminadamn)** — Financial Inclusion Platform: A secure full-stack platform (React, Node/Express, MongoDB, JWT) automating NBCFDC loan eligibility for 500+ simulated users, cutting manual processing time by 75%, backed by a KNN risk-banding ML pipeline. **Smart India Hackathon 2025 Finalist.**
4. **[RepoGraph](https://github.com/saminadamn/Repograph)**: A graph-based GitHub repository analyzer that cuts LLM token usage 50–70% by building a dependency graph of a repo and selecting only the most relevant files for a query, instead of dumping the full repo into context. FastAPI backend, AST-based graph builder, force-directed visualizer, CLI, and a 24-test suite.
5. **[Aurum AI](https://github.com/saminadamn/Aurum-AI)**: An autonomous enterprise cost intelligence system that analyzes operational data to detect cost inefficiencies, quantify financial impact, and recommend actionable optimizations. Built with Next.js, Tailwind CSS, and PapaParse for CSV-driven analysis. [Live demo](https://aurum-ai-eight.vercel.app).
